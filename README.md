Para rodar o projeto:

	Utilizar versão mais recente do Node v16+

	Rodar "Docker compose up" na pasta raiz 
	´´´
	 - git clone: https://github.com/KauanXleto/TesteAccenture.git
	 - cd TesteAccenture
	 - docker compose up

	 ´´´

	 Buildar TesteAccentureApi e TesteAccentureWeb juntos "Alterar nas configurações da solução para buildar os dois juntos"

	 Swagger: https://localhost:44373/swagger/index.html
	 Web: https://localhost:44309/
